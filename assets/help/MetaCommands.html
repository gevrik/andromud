Meta Commands

AndroMud also support a few meta commands.

1. Repeat command: #number followed by a simple/complex command.

For example: you can send "#3 n" and this is the same as typing "n" three times.

2. Wait command: #wa number, this will let the following commands wait a few milliseconds before sending. The exact time is specified in "num" part of this command.

For example: "n;#wa 2000;n" means send command "n" and wait for 2 seconds before sending the next "n"
